# NTC vs. TC Paper Experiments

This repository contains the experimental framework used to study
Non-Task-Coupled (NTC) and Task-Coupled (TC) coordination.

The code is organized around the theoretical components of the
framework rather than around individual applications.

Current experimental architecture:

- experiments/
- preference models
- cost functions
- configuration/action spaces
- NTC models
- TC models
- metrics
- plotting

Applications such as navigation are instantiations of these reusable
components.
